Django App Role
=========
Creates a basic Django Web App

Requirements
------------
There are no requirements for the role.

Role Variables
--------------
There are no variables needed for the role.

Dependencies
------------
You will need to have a Mysql Database created and a testdb database created

Example Playbook
----------------

    - hosts: webserver
      roles:
         - django_app
